A full-stack web application inspired by Airbnb.
Users can browse listings, filter by category, add reviews, and create/edit/delete their own listings.
Built with Node.js, Express, MongoDB, EJS, Multer & Cloudinary.

Live Demo: https://wanderlust-project-1-d0t4.onrender.com

Features:
Browse all property listings
Filter by categories (Trending, Rooms, Mountains, Camping, Farms, etc.)
User authentication & authorization
Add, edit, delete listings
Upload images using Multer + Cloudinary
Add and manage reviews
Flash messages & form validation
Responsive design with EJS
